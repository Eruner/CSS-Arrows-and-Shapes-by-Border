Arrows, Borders, Shapes, all for quick copy-paste

Basic Arrows:

.arrow-left {
	padding: 0px;
	display: inline-block;
	border: 36px solid white;
	border-right: 36px solid black;
}
.arrow-up {
	padding : 0px;
	display : inline-block;
	border : 36px solid white;
	border-bottom: 36px solid black;
}
.arrow-down {
	padding : 0px;
	display : inline-block;
	border : 36px solid white;
	border-top: 36px solid black;
}
.arrow-right {
	padding : 0px;
	display : inline-block;
	border : 36px solid white;
	border-left: 36px solid black;
}

You can combine arrows (triangles) together:


used CSS:
div{
	display: inline-block;
	font-size: 0;
}
.tri1{
	border-left: 100px solid transparent;
	border-top: 100px solid #CE5937;
}
.tri2{
	border-top: 100px solid transparent;
	border-right: 100px solid #CE5937;
}
.tri3{
	border-right: 100px solid transparent;
	border-bottom: 100px solid #CE5937;
}
.tri4{
	border-bottom: 100px solid transparent;
	border-left: 100px solid #CE5937;
}
used HTML:

<div>
	<div class="tri2"></div>
	<div class="tri3"></div><br/>
	<div class="tri1"></div>
	<div class="tri4"></div>
</div>



same CSS as above, and used this HTML:

<div>
	<div class="tri4"></div>
	<div class="tri1"></div><br/>
	<div class="tri3"></div>
	<div class="tri2"></div>
</div>

You can make rectangular base to achieve this effect:


used CSS:
div{
	display: inline-block;
	font-size: 0;
}
.tri11{
	border-left: 50px solid transparent;
	border-top: 100px solid #CE5937;
}
.tri22{
	border-top: 50px solid transparent;
	border-right: 100px solid #CE5937;
}
.tri33{
	border-right: 50px solid transparent;
	border-bottom: 100px solid #CE5937;
}
.tri44{
	border-bottom: 50px solid transparent;
	border-left: 100px solid #CE5937;
}
.down50{
	margin-top: -50px;
	transform: translateY(50px);
}

used HTML:

<div>
	<div class="tri44"></div>
	<div class="tri11 down50"></div><br/>
	<div class="tri33"></div>
	<div class="tri22"></div>
</div>



same CSS as above, and used this HTML:

<div>
	<div class="tri22"></div>
	<div class="tri33 down50"></div><br/>
	<div class="tri11"></div>
	<div class="tri44"></div>
</div>




you can combine two of them
together with negative margin
that you must calculate:

.move-left{
	margin-left: -150px;
}

used HTML:

<div>
	<div class="tri44"></div>
	<div class="tri11 down50"></div><br/>
	<div class="tri33"></div>
	<div class="tri22"></div>
</div>
<div class="move-left">
	<div class="tri22"></div>
	<div class="tri33 down50"></div><br/>
	<div class="tri11"></div>
	<div class="tri44"></div>
</div>
